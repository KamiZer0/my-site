Простой сайт с информацией о веб-разработке

И так. Ты решился начать изучать веб-программирование, но не знаешь с чего начать? Не беспокойся. Дядя Миша потратил пару дней на поиск всей нужной для тебя информации.

Немного ссылок на основную информацию

  • https://levelup.gitconnected.com/the-2020-web-developer-roadmap-76503ddfb327 Основа основ. На этом сайте на английском языке написаны все важные вещи которые должен знать веб разработчик в 2К2К алло ебать будущее уже наступило. По факту там даны ссылки и названия тех вещей которые ты должен начать учить. РИКАМИНДУЮ!!1
  • https://habr.com/ru/post/495800/ Небольшой пост в котором уже на русском языке просто и кратко написано что нужно учить. По сути повторяет информацию с предыдущего сайта но уже точечно.
  • http://htmlbook.ru/ Сайт с инфой по хтмл и цсс. Там же есть простенькие учебники и гайды.
  • https://htmlbase.ru Хорший сайт с инфой и задачками по хтмл и цсс.
  • Книга “Изучаем HTML, XHTML и CSS от Эрика Фримена”. Хорошая книжка для совсем основ.
  • Книга Learn Enough HTML to Be Dangerous An introduction to HTML тоже хорошая для основ на английском.
  • Книга HTML5 и CSS3 Разработка сайтов для любых браузеров и устройств пока не читал, но отзывы хорошие.
  • https://htmlacademy.ru/courses есть отличные тренажоры по многим вещам.
  • https://ru.code-basics.com/ тоже всякие задания и тд.
  • Книга Morgan N JavaScript для детей - пока не читал, но хорошие отзывы.
  • https://learn.javascript.ru/intro Основной учебник который ты должен штудировать по жаваскрипту.

Если хуе-мое просто все написать, то ты должен сразу нырять в хтмл+цсс+джаваскрипт+реакт. Потому что все работает на них. По факту это самый простой способ войти во фронтэнд (то как выглядит сайт). Есть еще бэкэнд (то как работает сайт). Хтмл довольно простой язык, который будет ясен после прочтения книги или даже пары гайдов (также в конце есть ссылки на видеокурсы на ютюбе по хтмл, очень хорошие). Все это закрепляется практикой. Затем браться нужно за цсс. Тут, на мой взгляд, чуть сложнее чем хтмл, но эти вещи все работают в паре. Затем уже нужен джс и, судя по вакансиям, реакт.

Но самое главное тут что? То что если собрался пиздовать в программирование, то надо начинать все самому ручками искать информацию для себя. По ссылкам даны все нужные тебе пункты. Вбиваешь в гугл название, добавляешь к запросу ИЗУЧИТЬ/КУРС НА ЮТЮБЕ/КНИГА/ТУТОРИАЛ ОТ ШКОЛЬНИКА В НОУТПАДЕ НА ЮТЮБЕ и дрочишь информацию. Впитываешь.

Также после того как получил хоть какие-то знания, можно таки пройти курсы по программированию от какой-либо академии по интернету:

  • https://skillbox.ru/course/frontend-developer/
  • https://geekbrains.ru/professions/frontend_developer
  • https://htmlacademy.ru/profession/frontender

Либо какой-то курс на твое усмотрение. Это конечно по желанию, но многие из этих курсов можно стащить с рутрекера, просто набрав html academy и там все будет. Там же посоветовал бы скачать какой-нибудь курс по FIGMAе (редактор графический, который сейчас почти всегда в веб разработке используется) ну и возможно курсики по фотошопу.

Путь нелегкий, но ты справишься. И не забывайте что я сам хуй в стакане и только вкатываюсь в эти вещи.

Информация собранная со всех уголков

Халеварная тема, что проще... крч делай так:
1) учи нативный js (всегда пригодятся знания), без библиотек, чисто js
2) параллельно читаешь книги по алгоритмам и структурам данных
3) потом берешь библиотеки react, vue ( react - больше вакансий, vue - проще для понимания)
4) учи typescript, и смежные библиотеки, если реакт учить будешь, то это mobx, redux, rout
5) учи node ( именно серверную часть, самый минимум, напиши api простой)
6) ssr (next.js, если реакт выбрал)
Заведи гитхаб репозиторий и кидай туда весь учебный мусор, гит кста обязательно осваивай сразу, он не сложный, для понимания почитай про гитфлоу
После этих шагов пили партфолио, обязательно работа с формами, платежные фичи

Завтра ищешь в интернете книжку Dive into python. Похуй если ничего не поймешь. Затем идешь на python.org и изучаешь стандартную библиотеку от корки до корки. Потом зубришь, именно, сука, вызубриваешь конвенцию по написанию питоньего кода - PEP8, чтобы от зубов отскакивало. Когда напишешь свою первую имиджборду, по пути изучив верстку на html+css, скачиваешь и изучаешь любой питоний асинхронный вебсервер, рекомендую Tornado или Gevent. Как переделаешь имиджборду, чтобы выдавала по крайней мере 5 тысяч запросов в секунду, можешь идти дальше - тебя ждет увлекательный мир хайлоада. Apache Hadoop, сверхбыстрые асинхронные key-value хранилища, MapReduce. Отсос хиккующих выблядков / просто неудачников типа рейфага или сисярп/джава-хуесосов, которые сосут хуй по жизни не заставит себя ждать и уже через пол года ты будешь получать такие суммы, что любая баба будет течь при одном упоминании твоей зарплаты.
